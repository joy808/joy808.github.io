# Babel

자바스크립트 컴파일러

자바스크립트 표준 ECMAScript(ES)의 최신 문법으로 작성된 코드를, 실행 가능한 이전 버전 문법으로 변환해주는 트랜스파일러

최신 버전의 자바스크립트 문법을 모든 브라우저가 이해할 수 없기 때문에 해당 코드를 사용중인 브라우저에서 이해할 수 있는 문법으로 변환해준다.

ES6, ES7 등 최신 문법을 사용해서 코딩할 수 있기 때문에 생산성이 향상된다.
